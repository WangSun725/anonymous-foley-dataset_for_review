# Anonymous Dataset for Double-Blind Review

This repository contains the anonymized review version of a Foley audio dataset currently under double-blind peer review.

## Overview

This dataset is designed to support research on Foley sound analysis and related machine learning tasks. It contains 10,000 curated audio clips together with a metadata file describing the dataset organization and annotations.

To preserve anonymity during the review process, all author-identifying information has been removed from this repository.
